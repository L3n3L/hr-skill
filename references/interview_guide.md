# HR 面试评估指南

## 面试流程设计

### 初试 - 技术基础筛选
**目的**: 验证简历真实性 + 基础能力评估
**时长**: 30-45分钟
**形式**: 问答 + 基础实操

### 复试 - 深度能力考察
**目的**: 复杂问题解决 + 文化匹配
**时长**: 45-60分钟
**形式**: 案例分析 + 行为面试

### 终面 - 综合决策
**目的**: 高层对齐 + 最终确认
**时长**: 30分钟
**形式**: 战略思维 + 动机确认

## STAR法则提问模板

### Situation (情境)
- 请描述一个您参与的复杂项目
- 当时面临的主要挑战是什么？

### Task (任务)
- 您在这个项目中的职责是什么？
- 您的KPI指标是什么？

### Action (行动)
- 您具体采取了哪些行动？
- 为什么选择这种方式？

### Result (结果)
- 最终结果如何？
- 您的贡献占比多少？
- 有什么可以改进的？

## 关键能力面试题

### 1. 技术能力
```
- 请手写一个快速排序算法
- 讲解一个你最得意的项目架构
- 如何优化一个慢查询？
```

### 2. 问题解决能力
```
- 描述一次你解决线上紧急故障的经历
- 遇到技术方案无法按时完成怎么办？
- 如果团队出现技术路线分歧如何处理？
```

### 3. 沟通协作
```
- 如何向非技术人员解释技术问题？
- 与产品经理需求不一致时怎么处理？
- 跨部门协作遇到过哪些挑战？
```

### 4. 学习成长
```
- 最近学习了什么新技术？
- 如何保持技术竞争力？
- 职业发展的下一个目标是什么？
```
